\subsection{Eight surveyed sub-areas}

\begin{enumerate}
  \item Model cascading (FrugalGPT and successors)
  \item LLM routing (RouteLLM, Hybrid LLM Routing, AutoMix, Tryage)
  \item Confidence-based routing (semantic entropy, hidden-state probes,
        verbalized confidence)
  \item Speculative decoding (token-level only; not applicable at turn
        level — too expensive to reject a whole turn)
  \item Mixture-of-Experts at the orchestration layer
  \item Per-turn classification paradigms (semantic similarity,
        DistilBERT trained, LLM router)
  \item Cost-performance frontier benchmarks (SWE-bench, BigCodeBench,
        Aider Polyglot)
  \item Routing-eval methodology (regret, expected utility, Pareto
        frontier)
\end{enumerate}
