- Same as TRIM-05: print-safe colors, `width=0.85\textwidth,
  height=6cm` for consistency, `nodes near coords` for bar
  values.

  visual weight, e.g. a larger label font on the 99.4% slice.
- fig4 (composed-cost) is the punchline — explicit % labels on
  each step.
