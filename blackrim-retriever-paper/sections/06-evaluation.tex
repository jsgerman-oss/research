\subsection{Setup}

\paragraph{Trace.}
Per-query telemetry from \texttt{internal/bdmemory/paper\_stream.go}
flowing to \texttt{data/raw/queries.jsonl}: query hash, query length,
inferred query class, per-scorer top-$k$ scores and latency, final
fused ranks, and total latency. Trace window: \tothink{30}-day rolling.
Total queries: \tothink{N}; per-class breakdown
\tothink{$N_{\queryclass_1}, \ldots, N_{\queryclass_6}$}.

\paragraph{Relevance judgements.}
Per-query relevance judgements built via \tothink{describe the labeling
protocol --- pooled top-$k$ across baselines, judged by an LLM-judge
worker with spot-check against operator labels, plus a sample held-out
human-judged set for calibration. Discuss inter-rater agreement.}

\paragraph{Baselines.}

\noindent\textbf{Note on prototype vs.\ proposed stack.}  Measured
results use the current production prototype
($\scorers_{\text{prototype}} = \{\texttt{keyword}, \texttt{depgraph}\}$,
\S\ref{sec:system-prototype}).  Baselines requiring BM25, SPLADE, or
dense retrieval are marked \tothink{} below; they will be populated
once those scorers ship in \texttt{internal/bdmemory/}
(\S\ref{sec:system-proposed}).

\begin{itemize}[leftmargin=*]
  \item \textbf{Keyword} --- the historical \blackrim default (legacy
        weighted-overlap scorer; ships in prototype). Measured.
  \item \textbf{Keyword + depgraph-RRF} --- 2-arm RRF fusion over the
        current prototype scorers. Measured.
  \item \textbf{BM25} \tothink{} --- the canonical IR baseline; the
        per-class safety floor for $\queryclass_1, \queryclass_3,
        \queryclass_5, \queryclass_6$ in the proposed stack.
        Pending BM25 deployment.
  \item \textbf{BM25 + decay} \tothink{} --- per-class safety floor for
        $\queryclass_4$ (continuity) in the proposed stack.
        Pending BM25 deployment.
  \item \textbf{Hybrid-RRF (BM25+SPLADE+dense)} \tothink{} --- the
        literature default \cite{cormack2009reciprocal}.  Pending SPLADE
        and dense deployment.
  \item \textbf{Hybrid-CC (BM25+dense, $\alpha$ tuned per class)}
        \tothink{} --- Bruch et al.\ 2023 \cite{bruch2023analysis}
        prescription.  Pending dense deployment.
  \item \textbf{CCBandit-Retriever (ours)} --- \cref{alg:ccbandit}
        operating over $\scorers_{\text{prototype}}$. Measured against
        prototype scorers; to be re-evaluated against proposed stack.
\end{itemize}

\subsection{Ablation: removing the conservative gate}

\tothink{ablate Layer 2 (the conservative-bandit gate). Expected
finding: gate-disabled variant gains a small aggregate NDCG@10 lift
but introduces per-class regression on at least one of the Critical
classes ($\queryclass_2, \queryclass_6$) --- which is the design-intent
violation.}

\subsection{Ablation: removing the SPLADE corpus-size gate}

\tothink{ablate the corpus-size threshold that gates SPLADE engagement.

cost.}

\subsection{Reproducibility}
\label{sec:repro}

Every quantitative claim is backed by a CSV in \texttt{data/} produced
by a script in \texttt{scripts/}. The pipeline:

\begin{lstlisting}[language=bash]
python scripts/pull-telemetry.py --since=30d > data/raw/queries.jsonl
python scripts/aggregate-by-class.py < data/raw/queries.jsonl \
    > data/aggregated/by-class.csv
python scripts/run-eval-suite.py --policy=keyword       > data/aggregated/eval-keyword.csv
python scripts/run-eval-suite.py --policy=bm25          > data/aggregated/eval-bm25.csv
python scripts/run-eval-suite.py --policy=bm25-decay    > data/aggregated/eval-bm25-decay.csv
python scripts/run-eval-suite.py --policy=hybrid-rrf    > data/aggregated/eval-hybrid-rrf.csv
python scripts/run-eval-suite.py --policy=hybrid-cc     > data/aggregated/eval-hybrid-cc.csv
python scripts/run-eval-suite.py --policy=conservative-cb \
    > data/aggregated/eval-cb.csv
make
\end{lstlisting}

The pipeline is deterministic given fixed RNG seeds and a snapshot of
the production telemetry; we anchor the published version to a tagged
commit. The companion artefacts repository
\cite{blackrim_retriever_paper_repo} hosts the data and scripts.
