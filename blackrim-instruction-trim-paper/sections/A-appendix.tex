\subsection{Six categories surveyed}

\begin{enumerate}
  \item Token-efficiency wins beyond caching
        (speculative decoding, function-calling, structured outputs).
  \item Local-hardware exploitation
        (small LMs as drafters / classifiers; ONNX CPU embedding).
  \item Multi-agent coordination patterns
        (KV-cache sharing, hedged requests, batch optimisation).
  \item Math/CS/ML algorithms
        (Thompson sampling for dispatch, RLAIF, GNNs on bd DAG,
         control theory for cadence).
  \item Eval pre-generation for gating
        (template-based, mutation-driven, RLAIF-adapted).
  \item Orthogonal systems
        (cache-aware scheduling, prompt deduplication, append-only
         telemetry, gRPC streaming).
\end{enumerate}
